How to create your own repo ?

What's a repo ?

A repo is a json file that's contain apps to grow an existing library. This concept has been popularized with Cydia.

How to create your own repo ?

If you want to distribute your apps on iTweakHub, you can make your own repo for free ! Requirements :

  • A GitHub account
  • A direct link to the apps (.iPA files)
  • (optional) chatGPT to verify the good syntax of your repo

The syntax.

You must follow the syntax of the iTweakHub repos and base json syntax to avoid errors.
